GrammarVis

GrammarVis visualizes the syntactic structure of sentences in a foreign language, to help learners learn the grammar and word order. You can read about it here, or watch a demo video.

Try it

It can be used standalone (sample sentence), or as a Chrome Extension for viewing articles and manga (examples - install the extension first).
